# Buy Now Pay Later (BNPL)

**Company:** Flipkart ($40B Ecommerce Platform)
**Role:** Software Engineer 1, Payments
**Timeline:** Jul 2018 – May 2020

## Overview
Developed payment workflows and external lending-partner API integrations to launch Buy Now Pay Later — the first such offering on an Indian e-commerce platform.

## Impact
- Launched BNPL to **150M+ users**
- First BNPL product on an Indian e-commerce platform

## What I Did
- Developed end-to-end payment workflows for the BNPL product
- Built API integrations with external lending partners
- Collaborated with payments, risk, and business teams to ship the feature

## Skills & Tools
Payments, API Integration, BNPL, Lending, FinTech, System Design, Java/Python, REST APIs
