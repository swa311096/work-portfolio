# Home Page Pagination

**Company:** Dunzo (Google-backed Quick Commerce Platform)
**Role:** Software Engineer 2, User Experience
**Timeline:** Jun 2020 – Jan 2021

## Overview
Architected and implemented pagination for Dunzo's app home page to drastically reduce load time for millions of daily users.

## Impact
- Reduced home page load time by **75%**
- Improved experience for **2M daily users**

## What I Did
- Identified home page latency as a key UX bottleneck
- Designed and implemented a pagination architecture to lazy-load content
- Collaborated with UX and backend teams to ship the feature end-to-end

## Skills & Tools
System Design, Pagination, App Performance, Latency Optimization, Android/iOS, Backend APIs, Python
