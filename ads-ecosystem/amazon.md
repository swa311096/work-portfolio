# Amazon Ads Ecosystem

**Core Signal:** Verified purchase history + "bottom of funnel" shopping intent.

**Moat:** The largest repository of retail intent data and a fully closed-loop ecosystem (Search → Buy).

**Platform Mechanics:** 
- **Sponsored Products/Brands:** Search-based retail ads.
- **Amazon DSP:** Allows off-site targeting using Amazon's first-party data. 
- **Amazon Marketing Cloud (AMC):** A "secure, privacy-safe, cloud-based clean room" enabling SQL-based cross-channel attribution.

**Measurement & Attribution:** 
Amazon Attribution provides tag-based measurement for off-platform media that links to Amazon listings/stores, including via API.

**APIs & Partner Ecosystem:** 
Amazon provides Ads APIs for Sponsored Products and Sponsored Display. For more enterprise programmatic and centralized operations, Amazon provides the Amazon DSP API and the Amazon Marketing Cloud (AMC) API.

**Strengths:** 
Strongest commerce intent and closed-loop sales measurement for brands selling on Amazon; unique transactions + media signals and clean-room analytics via AMC; scalable onsite-to-offsite activation through DSP.

**Limitations:** 
Ecosystem advantages concentrate most in endemic retail advertisers; creative options for sponsored search can be limited vs broad social platforms; off-Amazon measurement still requires careful instrumentation and AMC adoption for deeper analysis.

**Primary Use-cases:** 
Retail search conquesting, PDP/category defense, new product launches, and full-funnel retail media + offsite retargeting/CTV through DSP supported by AMC analytics.
